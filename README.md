## Instructions

### Setup


```shell
nvm install
nvm use
```

Or:

```shell
fnm install
fnm use
```

To install and launch the project, run these commands:

```shell
yarn
yarn start
```


### Deployment

- To build and deploy, run the following commands:

```shell
yarn build
yarn deploy
```
